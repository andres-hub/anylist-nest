# Dev

1. Clone the project
2. Copy the ```env.template``` and rename to ```.env```
3. Run
```
yarn install
```
4. Up image the data base, need Docker desktop
```
docker-compose up -d
```
5. Run the backend the Nest
```
yarn start:dev
```
6. visit the site
```
localhost:3000/graphql
```
